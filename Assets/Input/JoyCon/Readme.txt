# JoyCon Support

This asset is used add support for individual bluetooth JoyCons to Unity.

## Connecting
Hold down the small button (sync) on the top of the controller for 5 seconds - this puts the controller into broadcasting mode.
Search for it in your bluetooth settings and pair normally.
To disconnect the controller - press the sync button. To reconnect - press any button on your controller.

## How To Use
There is 1 main input device called Switch JoyCon.
There are 2 sub-devices Switch JoyCon Left and Switch JoyCon Right.

However they are all Gamepads so you can just use Gamepads as usual Input and it will work fine.
Because the JoyCon doesn't have all buttons on a Gamepad some inputs are never used.

Take a look at the example Input Schemes for all Inputs On the JoyCons.

### Example Scene
The example scene uses the "Player Input Manager" component to spawn new players when they press a button.
To try this out try to press A Button on a JoyCon to join. You can use the analog stick to move around and lower botton to rotate (This Demo works with any Gamepad).
